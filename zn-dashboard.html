Google Sheets Integration

Step 1: Google Sheet Setup

1
Create a new Google Sheet and name it "ZN Orders". Add the following headers in the first row:
Order ID | Timestamp | Name | Phone | Email | City | Address | Payment | Items | Subtotal | Delivery | Total | Note | Status

Step 2: Apps Script Deploy (Gmail Emails Included)

Gmail — No EmailJS Needed!
This script uses MailApp (Google's built-in Gmail) to send emails automatically. No external service needed. Edit ADMIN_EMAIL at the top of the script before deploying.
1
Open your Google Sheet → Extensions → Apps Script
2
Change ADMIN_EMAIL to your Gmail address (line 6 of the script)
3
Click Deploy → New Deployment → Web App. Set "Execute as: Me" and "Who has access: Anyone". Copy the URL.
4
First time: Google will ask for permissions — click "Allow" for Gmail and Sheets access.

Step 3: Paste Web App URL

1
After deployment, copy the generated Web App URL and paste it below, then click "Save URL".
